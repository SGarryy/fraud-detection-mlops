# Data

The dataset used in this project is the **Credit Card Fraud Detection** dataset.

- Source: https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
- File: `creditcard.csv` (144MB — not tracked by git)
- Download it and place it in this folder before running any scripts

## About the dataset
- 284,807 transactions made by European cardholders in September 2013
- 492 frauds out of 284,807 transactions (0.17% — severely imbalanced)
- Features V1–V28 are PCA-transformed (anonymized)
- `Time`, `Amount`, and `Class` are the only original features
- `Class = 1` means fraud, `Class = 0` means legitimate
